VMware

 

Apa Itu VMware?

VMware adalah software virtualisasi yang bisa digunakan untuk membuat virtual machine. Sebenarnya nama VMware diambil dari nama perusahaan pembuat software tersebut yaitu VMware, Inc.

 

Apa Kegunaan Vmware?

VMware memungkinkan kamu untuk membuat virtualisasi server, komputer, sistem operasi, storage device, aplikasi, networks, dsb.

Jadi bisa dibilang dengan VMware ini kamu bisa membuat komputer virtual di dalam komputer fisik, serta menjalankan sistem operasi didalam sistem operasi lainnya.

Di area server, virtualisasi VMware bisa digunakan untuk memaksimalkan resource hardware server memecah satu server fisik menjadi beberapa server virtual yang fungsional.

Di area penggunaan sehari-hari, virtualisasi VMware bisa kita manfaatkan untuk menjalankan sistem operasi di dalam sistem operasi, misalnya menjalankan Linux di dalam Windows (dan sebaliknya). Selain untuk keperluan testing, ujicoba OS, kita juga bisa memanfaatkannya untuk membuat sistem yang terisolasi dan tidak merusak sistem utama meskipun sistem di virtual machine tersebut kita oprek-oprek sesuka hati.

 

Cara Kerja Vmware?

VMware bekerja dengan membuat layer hardware virtual (virtual machine) di komputer kita (physical machine). VMware akan membuat storage virtual, CPU virtual, RAM virtual, DVD ROM virtual, dan berbagai komponen virtual lainnya untuk kemudian menjadi virtual machine yang utuh.

Layer software yang disebut dengan hypervisor itulah yang secara dinamis mangalokasikan resource di komputer fisik (host computer) ke virtual machine sesuai dengan yang dibutuhkan.

Begitu virtual machine tersebut diinstall sistem operasi, maka sistem operasi di virtual machine tersebut (guest OS) akan berjalan secara terisolasi di sistem operasi utama (host OS).

Setiap state dari virtual machine tersebut bakal tersimpan dalam bentuk file yang nantinya bisa kamu gandakan dan pindahkan dengan mudah.

Dengan begitu kamu bisa menjalankan beberapa virtual machine dengan beberapa spesifikasi dan sistem operasi hanya dari satu komputer fisik.

Untuk Virtualisasi Data Center dan Insfrastruktur Cloud

Jika yang kamu butuhkan adalah untuk keperluan virtualisasi di data center dan infrastruktur cloud, maka versi VMware yang tepat untuk kamu pilih adalah seri VMware vCloud, vSphere, dan vCenter.

 

Untuk Manajemen Data Center dan Cloud

Jika yang kamu butuhkan adalah memanage data center dan cloud dengan built in tools untuk keperluan otomasi dari manajemen VM, maka versi VMware yang tepat untuk kamu pilih adalah seri VMware vRealize

 

Untuk Infrastruktur as a Service

Jika yang kamu butuhkan adalah untuk implementasi hybrid cloud yang reliable dan aman di data center kamu, serta ingin mendeploy aplikasi aplikasi kamu secara cloud, maka versi VMware yang tepat untuk kamu pilih adalah seri VMware vClou Air.

 

Untuk Virtualisasi Desktop dan Aplikasi (Kelas Enterprise)

Jika kamu adalah IT admin dari perusahaan atau enterprise dan ingin memberikan Windows Desktop, aplikasi, ataupun layanan online kepada end user melalui virtual data center, virtual machine, ataupun physical device, maka yang kamu butuhkan adalah VMware seri Workspace Suite, Horizon, App Volumes, User Environment Manager, Mirage, ataupun ThinApp.

 

Untuk Aplikasi dan Data Platform

Bagi kamu yang punya aplikasi dan ingin di-skala melalui virtual infrastruktur, maka versi VMware yang tepat untuk kamu pilih adalah seri VMware Pivotal. Seri ini cocok sekali bagi kamu developer yang ingin menjadikan apps kamu bisa diakses melalui cloud dan bisa di skala dengan mudah.

 

Untuk Personal Desktop

Inilah versi VMware yang paling banyak digunakan untuk keperluan sehari-hari. Jika kamu hanya ingin membuat virtual machine di PC, baik itu untuk keperluan testing OS, coba-coba OS, membuat sistem terisolasi, ataupun menjalankan OS lain secara virtual, maka yang perlu kamu lihat hanyalah seri VMware Workstation.

VMware Workstation punya dua jenis yaitu Workstation Pro dan Workstation Player.

Versi Pro merupakan versi berbayar dengan berbagai jenis kelebihannya, sedangkan Workstation Player bisa kamu download dan gunakan gratis untuk keperluan personal.

Sebenarnya di kategori ini masih ada juga VMware Fusion, tetapi karena itu untuk Mac maka tidak kita bahas disini.

 

Mana yang Sebaiknya Dipilih untuk Penggunaan Rumahan?

Penggunaan rumahan disini artinya penggunaan sehari-hari, misalnya membuat virtual machine di laptop dan komputer untuk keperluan testing OS, testing software, membuat sistem terisolasi, menjalankan Linux dari dalam Windows, dan berbagai keperluan dasar lainnya.

Untuk keperluan tersebut VMware Workstation Player adalah yang paling tepat untuk kamu gunakan.

Selain bisa didapatkan secara gratis untuk keperluan personal, VMware Workstation Player ini sudah memiliki fitur yang lebih dari cukup untuk kebutuhan virtualisasi sehari-hari.
