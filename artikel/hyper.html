Virtualisasi Microsoft Hyper -V

Hypervisor bisa di ibaratkan suatu aplikasi khusus yang berjalan pada suatu server yang memungkinankan berbagai macam system operasi dapat berjalan diatasnya.

Secara umum hypervisor dibagi menjadi 2 jenis yaitu :

Baremetal Architecture yang dikenal sebagai hypervisor type 1, tipikial ini berjalan langsung diatas perangkat keras server, artinya tidak diperlukan system operasi lain untuk menjalankan hypervisor type 1 ini. Contoh yang paling kita kenal sekarang adalah VMware ESX/ESXi, Microsoft Hyper V.

Hosted Architecture yang dikenal sebagai hypervisor type 2, jenis ini memerlukan OS lain sebelum bisa digunakan. Jadi kita misalnya perlu menginstal OS Windows atau linux terlebih dahulu sebelum menginstal hypervisor type 2 ini. Contoh yang sering kita jumpai adalah VMware Workstation, Virtual Box, Virtual PC.
